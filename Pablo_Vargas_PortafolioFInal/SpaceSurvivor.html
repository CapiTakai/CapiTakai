Space Survivor

Space Survivor es una corta demo de un juego First-Person Shooter de ciencia ficción y terror. Toma como inspiración clásicos del medio como Dead Space, Bioshock o System Shock. Desarrollando con el motor gráfico de Unity Engine, este juego busca combinar el combate en primera persona con un ambiente misterioso y una narrativa ambiental que busca brindar una experiencia inmersiva dentro de este complejo científico espacial abandonado.

A pesar de ser una experinecia corta, se busca generar un ambiente tenso para el jugador e invitarlo a explorar el nivel. El entorno cuenta con luces, sonidos, efectos y animaciones que buscan reforzar la atmósfera, pasando de un área tranquila que parece simplemente abandonada, a otras que empiezan a mostrar que algo más pasó en este lugar. El trabajo de codificación de este proyecto se realizo con C#, asegurandose de que aspectos como el jugador o el sistema de armas sean modulares. Esto fue hecho con el objetivo de permitir al proyecto expandirse en un futuro de ser posible.

El desarrollo de este proyecto busca ser como se mencionó una demostración de las capacidades que se poseen para Game Development, trabajando en todos los aspectos de un juego. Yendo desde el desarrollo del nivel con bloking mediante la herramienta ProBuilder de Unity, como un boceto para organizar el nivel, para pasar a la construcción del mismo utilizar elementos de la Unity Asset Store. Pasando después a crear la ambientación del mismo, jugando tanto con la luz ambiental como con la iluminación del escenario en sí, a esto se le acompaña el uso de particulas y efectos de sonido.A la vez se trabajaron las interacciónes y mecanicas del mismo, como las animaciones e interacciones para abrir y cerrar las puertas del nivel, como una linterna, recolección de items y el sistema de disparos. Por ultimo se creó el sistema e inteligencia de los enemigos, siendo asi un proyecto bastante completo.
